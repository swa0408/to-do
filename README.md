# To-Do
Covey's matrix inspired To-Do list.

1. Idea :

Inspired by Stephen Covey’s Time Management Matrix.
Helps users categorize tasks into four quadrants:

🔴 Urgent & Important → Do First

🔵 Not Urgent but Important → Schedule

🟡 Urgent but Not Important → Delegate

🟢 Not Urgent & Not Important → Eliminate

Goal -> Build a personal productivity tool with a clean, responsive interface.


2. Technologies Used :
 
HTML5 → Structure of landing page and task list.

CSS3 → Styling, responsiveness, animations, color coding.

JavaScript (ES6) → Interactivity: add, delete, edit, tick tasks, and priority sorting.

Responsive Design → Flexbox & Grid for layout, mobile‑friendly UI.


3. Key Concepts :
   
Landing Page UX → Motivational welcome screen with Covey’s Matrix illustration.

Task Management → Add, tick (completed → grey + bottom), edit, delete.

Priority Sorting → Automatic order: Red → Blue → Yellow → Green.

Responsive UI/UX → Works seamlessly across devices.


4. Learning Outcomes :
   
Applied DOM manipulation and event handling in JS.

Practiced modular design (separate HTML, CSS, JS).

Understood UI/UX principles and responsive design.

Learned to implement a theoretical productivity model into a working app.
